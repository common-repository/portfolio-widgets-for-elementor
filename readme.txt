Elementor Portfolio Widget Plugin that offers the possibility to add Portfolio Widget to your Elementor WordPress Website

== Description ==

[youtube https://youtu.be/rcsQ2REMU_U]

Elementor Portfolio Widgets Plugin by Codeless comes with various styles and options. It's fully responsive and works perfectly with Elementor Plugin

You can choose from 1-2-3-4-5 columns, 2 different portfolio styles, advanced portfolio items query. It comes with a custom post type: "Portfolio Items"

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install "Elementor" & "Elementor Portfolio by Codeless" plugins
2. Add Custom Portfolio Items on Dashboard -> Portfolio Entries
3. Add Codeless Portfolio Grid Widget to the pages you need to showcase Items
4. Change Options and Save!
